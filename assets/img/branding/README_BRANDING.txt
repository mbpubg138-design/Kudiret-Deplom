ПАПКА ДЛЯ ЗАМЕНЫ ЛОГОТИПОВ И ИКОНОК

Меняйте файлы прямо в этой папке, чтобы логотип обновился на всём сайте.

Файлы:
- site-logo.svg      -> основной логотип сайта (шапка, футер, другие места)
- favicon.svg        -> иконка вкладки браузера
- phone-icon.svg     -> иконка рядом с номером телефона

Если хотите использовать PNG/JPG/WebP вместо SVG:
1) Положите новый файл в эту папку.
2) Откройте includes/config.php
3) Измените путь в константах SITE_LOGO_PATH / SITE_FAVICON_PATH / PHONE_ICON_PATH

Сейчас сайт уже настроен так, чтобы логотип менялся везде из одного места.
